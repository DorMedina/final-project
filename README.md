## General info

This project is my final project for the course and its an online shop.

## Technologies

Project is created with:

- npm: 8.1.2
- React: 17.0.2
- Nodejs: 16.13.1

## Setup

- To run this project, install it locally using npm:

```
$ cd ../lorem
$ npm install
$ npm start
```

- Import products.json to the mongo database.
